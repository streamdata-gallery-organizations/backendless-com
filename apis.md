description: 'Backendless is an enterprise BaaS platform that enables developers to
  create mobile and desktop apps without server-side coding. The platform is available
  as a service in the cloud or as a downloadable product for private deployments.
  Company vision is to provide a development and runtime environment for integrating
  mobile applications with back-office and enterprise systems including ERP, SCM and
  CRM. The core of the product includes the foundation of services required for the
  majority of the mobile applications. These include user management, relational data,
  push notifications, publish/subscribe messaging, geolocation, file service and video/audio
  recording and live streaming. Amongst key and unique features are app versioning
  support, video recording and live streaming on the server, filtering for publish/subscribe
  messages, hosting of websites and apps, flex/air clients support. The functions
  of services are available through robust API. Backendless APIs are available in
  the SDKs for the following client-side environments: JavaScript, Android, iOS, Windows
  Phone, Flex/AIR. All APIs are also available via REST. Pay-as-you-go pricing model
  includes a generous free tier with unlimited API calls.'
